## 技术栈

## 详细说明

> **基于 Spring Boot + MyBatis + Redis 的外卖点餐平台** | 后端开发
>
> Maven 多模块工程（3 个子模块），主要负责**管理端**功能接口开发，参与用户端部分接口开发。
>
> **项目规模**：66 个 Java 源文件，约 8,200 行代码，11 个 Mapper XML（530 行 SQL），11 张业务表（订单表 24 个字段）
> **接口数量**：18 个 Controller，72 个 API 接口（管理端 46 个 + 用户端 24 个 + 支付回调 2 个）
> **管理端模块**：员工管理、分类管理、菜品管理、套餐管理、订单管理、数据统计（Excel 导出）、店铺营业设置
> **用户端（参与开发）**：微信登录、菜品浏览、购物车、下单、地址管理

## 工程实践

> - **数据库设计**：11 张业务表，逻辑外键关联，订单表 24 字段覆盖完整订单生命周期（下单→支付→接单→配送→完成/取消）
> - **全局异常处理**：`@RestControllerAdvice` + `@ExceptionHandler` 统一拦截，区分业务异常与 SQL 约束冲突，解析 `Duplicate entry` 自动定位重复字段
> - **日志规范**：全项目使用 `@Slf4j` 注解式日志，关键节点（拦截器注册、自动填充、组件初始化）均有日志埋点
> - **AOP 自动填充**：自定义 `@AutoFill` 注解 + 反射，按 INSERT/UPDATE 操作类型自动注入 createTime、updateUser 等公共字段
> - **定时任务**：Spring Task 每分钟扫描超时未付订单（15 分钟）自动取消，每日凌晨自动完成配送中订单
> - **实时推送**：WebSocket 维护客户端 Session 池，新订单产生后主动推送商家后台
